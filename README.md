# Trabajo individual FII – Web personal

Pequeña web para la asignatura **Fundamentos de la Ingeniería Informática**.

## Estructura

- `docs/`
  - `index.html` – Página principal.
  - `css/styles.css` – Estilos comunes.
  - `public/about.html` – Sobre mí.
  - `public/topic.html` – Tema / herramientas.
  - `public/net.html` – Red de compañeros.
  - `public/degree.html` – Información del grado.
  - `public/fii.html` – Detalle de la asignatura FII.
  - `public/contact.html` – Formulario de contacto (sin envío real).
  - `images/` – Imágenes usadas en el sitio.

## Publicación

El sitio se publica con GitHub Pages usando la carpeta `docs` de la rama `main`.
