Herramientas que utilizo en el grado

En esta página muestro las principales herramientas que uso para estudiar, programar y organizarme.

Software y plataformas principales

Visual Studio Code

Editor de código que utilizo para programar.

Git y GitHub

Herramientas de control de versiones y repositorios remotos que uso para guardar el código.

Microsoft Office

Aplicaciones como Word, Excel y PowerPoint que uso para hacer documentos, hojas de cálculo y presentaciones de trabajos.

Canvas (UFV)

Plataforma que utilizamos en la UFV para acceder a materiales, entregar tareas, consultar calificaciones y comunicarnos con los profesores.

Perplexity (IA)

Asistente de inteligencia artificial que utilizo para resolver dudas, buscar información, estudiar y preparar mejor los trabajos.
